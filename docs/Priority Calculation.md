## How Priority is Calculated?         
   1. User Notes
   2. Urgent event 
   3. Due Date/Over Due 
   3. Milestone events / focus (e.g,: Summer training - Semester)
   4. Task Aging
   5. Behavioral Optimization Analysis
   6. User Docs (Long Term Planing and thoughts)

### Quantitative Metrics
1. Due Date/ Over Due
2. List Size (a list with more tasks shall have higher prioirity) 
3. Task Aging  (a task that is not with a close due date/over due, and high age shall be put first)
4. Behavioral Optimization Analysis
