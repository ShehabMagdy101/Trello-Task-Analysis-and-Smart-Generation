# Project Requirements
---
- R0: System shall show analytical dashboard of tasks
- R1: System shall send a daily analytics to the user on telegram
- R2: System shall generate a logical set of tasks based on analytics and user data
- R3: the user shall confirm the generated Task set or regenerate it atleast one and can go back to the pre generated set
- R3: The system shall show a reasoning why it generated this exact set
- R5: The system shall show the day task set as a check box tasks
- R6: System shall automatically update done tasks at 02:00 AM showing what tasks done
